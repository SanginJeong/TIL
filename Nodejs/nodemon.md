## nodemon

매번 새로고침 할 때마다 다시 node 를 실행하기 번거롭다.

nodemon을 사용하면 된다..

#### 설치

    npm install nodemon

#### 시작

    nodemon app.js

#### 에러

재설치

    npm install -g nodemon

그래도 해결안됨

에러메시지

    nodemon : 이 시스템에서 스크립트를 실행할 수 없으므로 ....
