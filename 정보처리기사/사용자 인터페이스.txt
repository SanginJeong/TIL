사용자 인터페이스 (UI: User Interface) : 사용자와 시스템 간의 상호작용이 원활하게 이뤄지도록 도와주는 장치

분야
- 정보 제공과 전달을 위한 물리적 제어
- 컨텐츠의 상세적인 표현과 전체적인 구성
- 모든 사용자가 편리하고 간편하게 사용하도록 하는 기능

CLI (Command Line Interface) : 명령과 출력이 텍스트 형태로 이루어짐 (명령프롬프트)
GUI (Graphical User Interface) : 마우스로 선택하는 그래픽 환경 인터페이스
NUI (Natural User Interface) : 사용자의 말이나 행동으로 기기 조작

UI의 원칙
-직관성
-유효성
-학습성
-유연성
